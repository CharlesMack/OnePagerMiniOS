Informational only; if possible call emergency services.

  1. Check scene safety. Protect yourself.
  2. Wear gloves/face covering if available.
  3. Check responsiveness (tap/shout).
  4. Airway, Breathing, Circulation (control severe bleeding).
  5. Call for help when available; send a bystander.
  6. If not breathing normally, begin CPR (hands-only if untrained).
  7. Place breathing but unresponsive person in recovery position.
